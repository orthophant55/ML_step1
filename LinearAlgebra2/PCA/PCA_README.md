# 선형대수학 응용 - 주성분분석(PCA)

## 1. 개념 요약

주성분분석(PCA, Principal Component Analysis)은 고차원 데이터를 더 낮은 차원으로 줄이는 대표적인 차원 축소 기법이다.

핵심 아이디어는 다음과 같다.

> 데이터를 어떤 방향으로 정사영했을 때 원래 데이터의 구조를 가장 잘 보존할 수 있을까?

PCA는 이 질문에 대해 **데이터의 분산이 가장 큰 방향**을 찾는다.  
이 방향을 **주성분(Principal Component)**이라고 한다.

---

## 2. PCA의 직관

2차원 데이터가 있다고 하자.

예를 들어 학생들의 국어 점수와 영어 점수가 있을 때, 두 점수를 하나의 종합 점수로 줄이고 싶을 수 있다.

이때 단순 평균을 사용할 수도 있지만, 데이터가 실제로 가장 많이 퍼져 있는 방향으로 점수를 만들면 원래 데이터의 구조를 더 잘 보존할 수 있다.
